Loot Object
	instantiable object
	stores the id of an item prototype
	interact to pick up

Item Object
	defines an object prototype
	base stats
	graphics and text
	effect object

Effect Object
	an instantiable object that carries out the effect of an item or ability

Item Registry
	array of item prototype objects
